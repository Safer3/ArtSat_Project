# ArtSat_Progect
Проект по цифровой практике
Концепт художественной визуализации
• CubeArtSat превращает параметры космического окружения — солнечный ветер, магнитные поля, радиационный фон — в элементы цифрового искусства.
• Цель: создать «космическую кисть», где реальные данные формируют цвет, свет, форму и движение изображения.
