# ArtSat_Progect
Проект по дисциплине "Цифровая практика"

1	ОСНОВНАЯ ИДЕЯ ПРОЕКТА
1.1	Концепт художественной визуализации
-	CubeArtSat превращает параметры космического окружения — солнечный ветер, магнитные поля, радиационный фон — в элементы цифрового ис-кусства.
-	Цель: создать «космическую кисть», где реальные данные формируют цвет, свет, форму и движение изображения.

1.2	Принципы художественного отображения
-	Цветовая палитра строится на активности Солнца: спокойствие - мягкие градиенты, бури - контрастные и резкие цвета.
-	Алгоритмические формы: линии потока, частицы.
-	Анимация отражает натуральную динамику космоса — медленные течения или бурные вспышки.
В итоговом результате проекта, предполагается получить программу, ко-торая на основе выбранной пользователем орбиты, выводит иллюстрацию, построенную на основе данных космических явлений.

1.3	Модули
	Работа над проектом предполагает разделение на 6 основных модулей:
-	Интерфейс 
Создание визуального отображения проекта.
-	Модель излучения 
Изучение околоземных явлений космического происхождения, которые можно тем или иным образом фиксировать и использовать в проекте.
-	Трансформация излучения в картинку 
Поиск, подходящих для проекта, способов преобразования данных в изображение.
-	Баллистика
Изучение космической механики для создания математической модели орбит, которые будут использоваться в программе.
-	DevOps
Организация рабочего процесса, системы контроля версий, работа с Git и github.
-	Общая логика
Логика работы функций, связь различных модулей программы.

ИНСТРУКЦИЯ ПО ЭКПЛУАТАЦИИ ПРОГРАММЫ

1. Выбор режима ввода данных
В левой верхней части окна программы (блок «Режим работы») выберите удобный для вас способ задания параметров орбиты:

По векторам состояния: Задание орбиты через точные координаты и скорости спутника в пространстве.

По орбитальным элементам: Задание орбиты через классические кеплеровы элементы (большая полуось, эксцентриситет, наклонение и т.д.).

(Примечание: при необходимости вы также можете вручную задать гравитационный параметр в поле «Гравитационный параметр μ»)

2. Ввод параметров
В зависимости от выбранного режима, заполните соответствующие поля в левой панели:

Координаты (x, y, z) в километрах и Скорость (vx, vy, vz) в км/с.

ИЛИ Орбитальные элементы: большая полуось (a), эксцентриситет (e_val), наклонение (i_deg), долгота восходящего узла (Ω), аргумент перицентра (ω) и истинная аномалия (v).

Если у вас есть заранее подготовленный файл с данными, вы можете воспользоваться кнопкой «Загрузить файл» в правом нижнем углу экрана.

3. Выполнение расчета
После ввода данных нажмите синюю кнопку «Рассчитать» в самом низу левой панели.
Программа автоматически вычислит недостающие значения и заполнит информационные блоки:

Рассчитанные параметры орбиты / векторы состояния (в зависимости от того, что вы вводили).

Блок «Орбита» с расчетами высоты перицентра, апоцентра и соответствующих расстояний.

4. Режимы отображения (Визуализация)
В правой части экрана (блок «Отображение») вы можете переключаться между двумя режимами визуализации результатов с помощью соответствующих переключателей:
Режим «Орбита»: Открывает интерактивную 3D-модель орбиты. В этом режиме вы можете наглядно проверить правильность заданных параметров, визуально оценить форму рассчитанной орбиты и положение спутника в пространстве.  Режим
«Визуализация» (Художественный режим): Переключает экран в режим генерации цифрового искусства, где параметры космической среды преобразуются в динамическую абстрактную визуализацию в реальном времени. Картина напоминает потоки жидкости, которая плавно реагирует на изменения входных данных и никогда полностью не повторяется.
