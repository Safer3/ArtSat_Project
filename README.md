# ArtSat_Progect
Проект по дисциплине "Цифровая практика"
1	ОСНОВНАЯ ИДЕЯ ПРОЕКТА
1.1	Концепт художественной визуализации
-	CubeArtSat превращает параметры космического окружения — солнечный ветер, магнитные поля, радиационный фон — в элементы цифрового ис-кусства.
-	Цель: создать «космическую кисть», где реальные данные формируют цвет, свет, форму и движение изображения.

1.2	Принципы художественного отображения
-	Цветовая палитра строится на активности Солнца: спокойствие - мягкие градиенты, бури - контрастные и резкие цвета.
-	Алгоритмические формы: линии потока, частицы.
-	Анимация отражает натуральную динамику космоса — медленные течения или бурные вспышки.
В итоговом результате проекта, предполагается получить программу, ко-торая на основе выбранной пользователем орбиты, выводит иллюстрацию, построенную на основе данных космических явлений.

1.3	Модули
	Работа над проектом предполагает разделение на 6 основных модулей:
-	Интерфейс 
Создание визуального отображения проекта.
-	Модель излучения 
Изучение околоземных явлений космического происхождения, которые можно тем или иным образом фиксировать и использовать в проекте.
-	Трансформация излучения в картинку 
Поиск, подходящих для проекта, способов преобразования данных в изображение.
-	Баллистика
Изучение космической механики для создания математической модели орбит, которые будут использоваться в программе.
-	DevOps
Организация рабочего процесса, системы контроля версий, работа с Git и github.
-	Общая логика
Логика работы функций, связь различных модулей программы.
